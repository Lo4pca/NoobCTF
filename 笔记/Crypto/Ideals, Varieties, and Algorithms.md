### Parametrizations of Affine Varieties

1. 写出线性方程组所有解的参数形式
- x + 2y − 2z + w = −1(1)
- x + y + z − w = 2(2)

这类题怎么做可参考第29页，用行变换消去变量。首先相加两个原始方程：2x+3y-z=1,得z=2x+3y-1。代回(1)，得x+2y+2(2x+3y-1)+w=5x+8y+w=1(3)；代回(2),得x+y+(2x+3y-1)-w=3x+4y-w=3(4)

在(3)和(4)上故技重施，使两倍的(4)减去(3)，得x-w=5,x=5+w。代回(4)，得3(5+w)+4y-w=2w+4y=-12， $y=\frac{-12-2w}{4}$ 。让w等于参数t，则：
- x=5+t
- $y=\frac{-12-2t}{4}=-\frac{t}{2} - 3$
- $z=2(5+t)+3(\frac{-12-2t}{4})-1=\frac{t}{2}$
- w=t

7. 改写第6题的论证，对 n 维仿射空间中的“球面” x₁² + … + xₙ² = 1 进行参数化。提示：将有 n − 1 个参数

设球的顶点为 $P_0=(0,...,1)$ 。固定 $x_n=0$ 并任取其他的坐标，得 $P_1=(x_1',...,0)$ ，然后连接两点。该线段与球的交点为 $(x_1'',...,x_n'')$

参数化得到的线段： $P_0+t(P_1-P_0)=(tx_1',...,1-t)$ ( $P_1-P_0$ 是代表“直线方向”的n维向量，所以这个式子表示“从 $P_0$ 出发，沿着直线方向走t个距离”)

然后算这条直线与球面的交点。将线的参数化形式代入球方程，求解t来找到直线与球面相交时的参数：
- $(tx_1')^2+...+(1-t)^2=1$
- $t^2(x_1'^2+...+x_{n-1}'^2)+1-2t+t^2=1$
- $t^2(x_1'^2+...+x_{n-1}'^2+1)-2t=0$
- $t(t[x_1'^2+...+x_{n-1}'^2+1]-2)=0$
- t=0或 $t=\frac{2}{x_1'^2+...+x_{n-1}'^2+1}$

将这个t代入直线的参数化方程，得： $x_i''=\frac{2x_i'}{x_1'^2+...+x_{n-1}'^2+1},\forall i$ < n， $x_n''=1-\frac{2}{x_1'^2+...+x_{n-1}'^2+1}=\frac{x_1'^2+...+x_{n-1}'^2-1}{x_1'^2+...+x_{n-1}'^2+1}$

8. 考虑由 y² = cx² − x³ 定义的曲线，其中 c 为某常数。我们的目标是参数化这条曲线

a. 证明一条直线与此曲线相交于 0、1、2 或 3 个点。提示：设直线方程为 x = a 或 y = mx + b

对于竖直线x=a,代入曲线方程 $y^2 = c a^2 - a^3$ 可得：
- $ca^2 - a^3 > 0$：$y = \pm \sqrt{ca^2 - a^3}$，2个交点
- $ca^2 - a^3 = 0$：$y = 0$，1个交点
- $ca^2 - a^3 < 0$：无实数解，没有交点

对于非竖直线y=mx+b，代入曲线方程 $(mx + b)^2 = c x^2 - x^3$ 并展开可得 $x^3 + (m^2 - c)x^2 + 2mbx + b^2 = 0$ 。这是一个三次方程，计重数的情况下在复数范围内一定有三个根且一定有一个实根，所以可能有1-3个交点

综合来看，可能的交点数为0-3

b. 证明一条过原点的非垂直直线，当 $m² \not= c$ 时，恰好与曲线交于另一点

已知三次方程的一个根为x=0，代入得 $b^2=0$ 。则方程化简为 $x^3 + (m^2 - c)x^2 = 0$ 。这个方程的判别式恰好为0，一定有重根。进一步分解得到 $x^2(x + (m^2 - c)) = 0$ 。只要 $m² \not= c$ ，方程就有两个根

c. 现在画垂直线 x = 1。给定该直线上一点 (1, t)，画出连接 (1, t) 与原点的直线。该直线与曲线交于点 (x, y)。从几何角度论证这给出了整条曲线的参数化

由于过原点的非垂直直线一定会与曲线上的另一点相交，任意不过原点的垂直直线（不一定要x=1）上一点(1, t)决定了直线的斜率。 $m^2=c$ 时给出的点为原点

9. 环索线（Strophoid）是一条被多位数学家研究过的曲线，其三角参数化由下式给出：
- x = a sin(t)
- y = a tan(t)(1 + sin(t))

其中 a 为常数

a. 求出描述该环索线的关于 x 和 y 的方程。提示：若不仔细处理，会得到方程 (a² − x²)y² = x²(a + x)²。要理解为何这不完全正确，请考虑 x = −a 的情况

三角参数化方程转关于x和y的方程的关键是利用三角恒等式消去参数。首先：
- $\sin(t)=\frac{x}{a}$
- $\tan(t)=\frac{y}{a(1 + \sin(t))}=\frac{y}{a(1+\frac{x}{a})}$

代入恒等式 $\sin^2(t)=\frac{\tan^2(t)}{1+\tan^2(t)}$ 得 $\frac{x^2}{a^2}=\frac{\frac{y^2}{a^2(1+\frac{x}{a})^2}}{1+\frac{y^2}{a^2(1+\frac{x}{a})^2}}$ ，化简后是 $\frac{x^2}{a^2} = \frac{y^2}{(a + x)^2 + y^2}$

这个定义和提示给出的“错误方程”其实是一样的，不过有理式额外说明了x不能等于-a

b. 求该环索线的代数参数化

有个叫万能代换（Weierstrass substitution）的东西：设 $u=\tan(\frac{t}{2})$ ，则 $\sin(t)=\frac{2u}{1+u^2},\tan(t)=\frac{2u}{1-u^2}$ 。所以：
- $x=\frac{2au}{1+u^2}$
- $y=-\frac{2 a u (u + 1)}{(u - 1) (u^2 + 1)}$

10. 大约公元前 180 年，狄俄克勒斯撰写了《论取火镜》(On Burning-Glasses)一书，他考虑的曲线之一是蔓叶线(cissoid)。他用此曲线解决了倍立方问题 [见下文 (c) 部分]。蔓叶线方程为 y²(a + x) = (a − x)³，其中 a 为常数

a. 求蔓叶线的代数参数化

观察过定点的旋转直线与曲线的交点是常用的代数参数化方法。曲线上明显的一点是(a,0)，则过该点的非竖直线的方程为y=t(x-a)。代入曲线方程： $(t(x-a))^2(a+x)=(a-x)^3$ ,得x=a（二重根）和 $x = \frac{a - a t^2}{t^2 + 1}$ ,则 $y=-\frac{2 a t^3}{t^2 + 1}$

b. 狄俄克勒斯用如下几何构造描述蔓叶线。给定半径为 a 的圆（以原点为圆心），在 a 与 −a 之间取 x，作直线 L 连接 (a, 0) 与圆上的点 $P = (−x, \sqrt{a² − x²})$ 。由此确定 L 上的点 Q = (x, y)。证明蔓叶线即为所有此类点 Q 的轨迹

需要证明点Q的坐标满足蔓叶线的方程。先找L的斜率： $\frac{\sqrt{a² − x²}}{-x-a}$ ，然后用点斜式得出直线方程： $y=\frac{\sqrt{a² − x²}}{-x-a}(X-a)$ 。代入Q的坐标x，得 $y=\frac{\sqrt{a² − x²}}{-x-a}(x-a)$ (所有出现的x都是一样的，看书中给出的实例图就能发现选定的x等于点Q的坐标x)。两边同时平方， $y^2=\frac{(a - x)^3}{a + x}$ ,符合题目给出的蔓叶线方程

c. 倍立方是经典希腊问题，即尝试用尺规作图构造 ³√2。已知仅用尺规这是不可能的。狄俄克勒斯证明，若额外允许使用蔓叶线，则可构造 ³√2。方法如下：作连接 (−a, 0) 与 (0, a/2) 的直线。此直线与蔓叶线交于点 (x, y)。证明 $2 = (\frac{a − x}{y})^3$ ，即可说明如何用尺规和蔓叶线构造 ³√2

直线的斜率是 $\frac{1}{2}$ ，所以方程是 $y=\frac{1}{2}(x+a)$ 。代入曲线方程： $(\frac{1}{2}(x+a))^2(a + x) = (a − x)^3\Rightarrow\frac{1}{4} (a + x)^3 = (a - x)^3$ ;同时 $y^3=\frac{1}{8}(x+a)^3$ ，利用 $(a+x)^3=4(a-x)^3$ 可得 $y^3=\frac{1}{2}(a-x)^3\Rightarrow 2=(\frac{a-x}{y})^3$

尺规作图可以作出长为任意两个线段之间的比例的线段，因此可作出a-x和y之间的比例 $\sqrt[3]{2}$
