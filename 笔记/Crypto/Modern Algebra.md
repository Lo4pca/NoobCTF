4. 代数证明以下命题：

(a) 若 H 是群 G 的一族子群，则 $\cap_{H_α ∈ H} H_α$ 也是 G 的子群

- 每个子群一定包含单位元，因此所有子群的交集也会包含单位元
- $\forall a\in H_α$ ，子群保证了 $a^{-1}\in H_{\alpha}$ ；意味着如果a出现在所有子群中, $a^{-1}$ 也是
- 同理， $\forall a,b\in H_α$ ， 子群保证了 $ab\in H_{\alpha}$ ;意味着如果a和b都出现在所有子群中, ab也是

综上所述，任意多个子群的交集，结果一定还是 G 的一个子群

(b) 对任意（可能为无穷的）子集 S ⊆ G，由 S 生成的子群定义为⟨S⟩ := { $s_1^{e1} s_2^{e2} · · · s_k^{ek} | s_i ∈ S, e_i$ ∈ {−1, 1}}。即 ⟨S⟩ 由所有能用 S 中元素及其逆写出的有限“字（words）”组成。注意 $s_i$ 无需互异。证明 ⟨S⟩ = $\cap_{S ⊆ H_α ≤ G} H_α$ ，其中交集取遍 G 的所有包含 S 的子群 [提示：要证 A = B，需证 A ⊆ B 且 B ⊆ A]

称 $\cap_{S ⊆ H_α ≤ G} H_α$ 为A。利用(a)的结论可以得出A是一个包含S的子群，自然 $⟨S⟩\subseteq A$

另一个方向需注意⟨S⟩的定义没有限制k的值，⟨S⟩允许长度为1的字, $S\subseteq ⟨S⟩$ ，⟨S⟩本身就是一个包含S的子群 $H_α$ ，因此所有 $H_α$ 的交不会大于⟨S⟩， $A\subseteq⟨S⟩$

5. 对于子群 H ≤ G 及元素 x ∈ G，集合 xH := { xh | h ∈ H } 称为 H 的一个左陪集

(a) 证明：若 x ∈ H，则 xH = H。从凯莱图的角度解释这一命题的含义

若 x ∈ H，则x与H中的其他元素h做运算时，因H是群，xh一定也落在H中

从凯莱图看，这等于说“将H的结构拷贝到x处”。由于x就是H所在的位置，这得到的仍是原本的H

(b) 证明：若 b ∈ aH，则 aH = bH

bH := { bh= $ah_ah=ah'$ | h ∈ H }=aH

(c) 证明：对任意 x ∈ G，映射 ϕ: H → xH，ϕ: h ↦ xh 是双射。由此推断 H 的所有左陪集大小相同

单射性： $xh_0=xh_1\Rightarrow h_0=h_1$

满射性直接从定义就能看出来了。单射+满射=双射，这告诉我们H和xH的大小相同，无论选定的x是什么

(d) 推断 G 被 H 的左陪集划分，且所有左陪集大小相等

左陪集之间没有交集，因为如果 $ah_1\in aH$ 且 $bh_2\in bH$ 有 $ah_1=bh_2$ ，则 $a=bh_2h_1^{-1}$ ，a在bH里。通过(b)可得aH=bH。由此可得G 被 H 的左陪集划分

“所有左陪集大小相等”在(c)证过了

2. f(x) = xⁿ − 1 的根是 n 个复数 $C_n$ := { $e^{2kπi/n} : k = 0, 1, …, n−1$ }，称为 n 次单位根。本原单位根是指满足 gcd(n, k) = 1 的 ζ = $e^{2kπi/n}$ 。容易看出 Q(ζ) 是 xⁿ − 1 的分裂域

(b) 证明：对任意本原单位根 $ζ_k = e^{2kπi/n}$ ，映射 $φ_k : Q(ζ) → Q(ζ)，φ_k(ζ) = ζᵏ$ 是一个域自同构。即，证明 $φ_k$ 是满的域同态（每个非零域同态都是单射）。若 gcd(n, k) ≠ 1，为什么这不是自同构？

可以把n次单位根看作是单位圆被平均分成的n个部分，任意部分旋转n次（旋转的角度等于该部分与正实轴的夹角）都可以回到1，但只有与n互质的部分在旋转过程中可以生成所有n个部分（循环群 $C_n$ ），即本原单位根

由于f(x)的次数是n，没法像`HW 11`的第4题那样写出域元素具体的结构来证明同态，需要用域元素的一般形式：Q(ζ)={ ${f(ζ)|f(x)\in Q[x]}$ }，即系数在 Q 中的所有多项式在 ζ 处求值的集合。则 $\phi_k:Q(ζ) → Q(ζ),\phi_k(f(ζ))=f(ζ^k)$

首先要证明映射的良定义：假设f(ζ)=g(ζ),需要证明 $f(ζ^k)=g(ζ^k)$ 。定义h(x)=f(ζ)-g(ζ),不难发现h(ζ)=0，即h(x)是ζ的零化多项式。ζ的极小多项式是 $\Phi_n(x)$ (以所有本原n次单位根为根的首一多项式。题目给出的f(x)可约，因此不是极小多项式)，根据极小多项式的基本定理，“若m(x)是a的极小多项式，则对任意零化多项式h(x)（h(a)=0），有m(x)|h(x)”。所以 $\Phi_n(x)|h(x),h(x)=\Phi_n(x)q(x)$ 。应用 $\phi_k$ 后， $f(ζ^k)-g(ζ^k)=\phi_k(h(ζ))=h(ζ^k)=\Phi_n(ζ^k)q(ζ^k)$ 。由于 $ζ^k$ 也是本原单位根， $\Phi_n(ζ^k)=0$ ，所以 $f(ζ^k)-g(ζ^k)=0$ ，映射良定义

接下来看是否保持加法和乘法：
- $\phi_k(f(\zeta)+g(\zeta))=\phi_k((f+g)(\zeta))=(f+g)(\zeta^k)=f(\zeta^k)+g(\zeta^k)=\phi_k(f(\zeta))+\phi_k(g(\zeta))$
- $\phi_k(f(\zeta)g(\zeta))=\phi_k((fg)(\zeta))=(fg)(\zeta^k)=f(\zeta^k)g(\zeta^k)=\phi_k(f(\zeta))\phi_k(g(\zeta))$

最后是满射。由于Q(ζ)中的任意元素都是ζ的多项式，只需要验证ζ本身是否在像中。利用前提条件的gcd(n, k)=1，可得存在 $a\in N$ 使得 $ak\equiv 1\mod n$ 。因此 $\phi_k(\zeta^a)=\zeta^{ak}=\zeta,\zeta$ 是 $\zeta^a$ 的像。由此， $im(\phi_k)=Q(\zeta),\phi_k$ 是满射

在gcd(n, k)不为1时， $\zeta$ 的阶是n/gcd(n,k) < n， $\phi_k(Q(\zeta))$ 的维数小于Q(ζ)，不可能是域自同构

3. 对下列每个多项式，判断其是否不可约。若不是，请将其分解为不可约因式的乘积

(a) 在 Q 上的f(x) = x⁴ − 10x³ + 12x² − 8x + 6

用248页的`Theorem 10.4` (Eisenstein Criterion)：2整除除1外的所有系数，但4不整除6，因此这个多项式不可约

(b) f(x) = x⁴ + x³ + x² + x + 1 [提示：令 u = x + 1，换元]

不确定提示是不是给错了。跟着提示换元的话， $f(u-1)=u^4-3u^3+4u^2-2u+1$ ，看不出来什么东西。但是如果换个符号，让u=x-1，则 $f(u+1)=(u+1)^4+(u+1)^3+(u+1)^2+u=u^4 + 5 u^3 + 10 u^2 + 10 u + 3$ ，明显满足Eisenstein Criterion的素数是5，因此f(x)不可约

(c) 在 Q 上的f(x) = x⁵ − 1

显然1是根，因此 $f(x)=(x-1)(x⁴ + x³ + x² + x + 1)$ 。后面这个多项式在上一小题已知不可约，因此这就是f(x)的分解

(d) 在 Q 上的f(x) = x⁶ − 1 [提示：搜索“分圆多项式”（cyclotomic polynomial）]
