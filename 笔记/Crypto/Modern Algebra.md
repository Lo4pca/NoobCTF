### Exercise 1.1

将一枚 1 美分硬币和一枚 5 美分硬币并排放在桌子上。仅考虑一个操作：交换这两枚硬币的位置。这是一个群吗？（逐一检查四条规则是否适用于此情况，并解释你的结论）

逐个检查四条规则（`Rule 1.5`-`Rule 1.8`）：
- 唯一的操作是“交换硬币的位置”。该操作不变且是提前定义好的
- 连续交换两次硬币的位置即可逆向原操作
- 交换硬币的操作是确定的
- 可以连续交换硬币无数次

四条规则都满足，因此它是群

### Exercise 1.4

(a) 计算重新排列这些画作的方式有多少种，前提是每面墙上只挂一幅画

3! = 6

(b) 考虑两个操作：你可以将左墙上的画与中间墙上的画互换，也可以将中间墙上的画与右墙上的画互换。仅凭这两个操作能否生成你刚才计算出的所有排列方式？

可以。因为这两个操作可以构造出另一个操作：将左右墙上的画互换。这样就等于刚才计算排列时没有任何限制的前提了

```py
n = 4
G = CyclicPermutationGroup(n)
r = G.gen()
C = G.cayley_graph()
vertex_labels = {g: 'e' if i==0 else f'r^{i}' if i==1 else f'r^{i}' for i, g in enumerate(G)}
P = C.plot(
    vertex_labels = vertex_labels,  # 使用自定义标签
    color_by_label = True,          # 边着色区分生成元
    edge_labels = True,             # 显示边的标签
    vertex_size = 0.1,
    vertex_color = 'lightblue',
    layout = 'circular',            # 使用圆形布局
    figsize = [4,4]
)
P.save('cayley_cn.png')
```
